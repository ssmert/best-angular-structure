# 브랜치

운영 : master

개발 : develop

# 설치

```shell
npm i
```

# 실행

```shell
로컬 : npm start
개발 : npm run start:dev
운영 : npm run start:prod
```

# 빌드

빌드 폴더 : dist/

```shell
로컬 : npm start
개발 : npm run build:dev
운영 : npm run build:prod
```
